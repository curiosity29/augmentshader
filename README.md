## How to start using (currently for window only):
 + Download and extract the zip file in Builds/
 + Run the augment_shader_v0.1.exe

## How to use:
+ Pick input and output folder
+ change the minimum and maximum value range for each shader parameter, value will be randomly taken in this range
+ press start augmenting and check out the progress tab
